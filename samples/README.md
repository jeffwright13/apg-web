# Sample Audio Files

This directory contains sample background audio files that users can select without needing to upload their own files.

## Required Files

Place the following MP3 files in this directory:

1. **sample1.mp3** - Meditative acoustic guitar
2. **sample2.mp3** - Upbeat background music
3. **sample3.mp3** - Mellow, happy, major pentatonic tune with Asian instrumentatation. "Lunar New Year" by Alex-Productions (https://freemusicarchive.org/music/alex-productions/single/lunar-new-year/).
4. **sample4.mp3** - Nature sounds (pines rustling in soft wind)
5. **sample5.mp3** - Nature sounds (birds on flowing river)

## File Requirements

- Format: MP3 (for smaller file sizes)
- Recommended duration: 30-120 seconds (will loop if needed)
- Recommended bitrate: 128-192 kbps
- File size: Keep under 2MB each for faster loading

## Copyright Considerations

Ensure all sample audio files are either:
- Original compositions
- Licensed for use (Creative Commons, royalty-free, etc.)
- Public domain

Recommended sources for royalty-free music:
- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/music/royalty-free/)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary)
- [ccMixter](http://ccmixter.org/)
- [Bensound](https://www.bensound.com/)

## Usage

Once files are in place, users can select them from the "Background Sound" dropdown in the app interface.
